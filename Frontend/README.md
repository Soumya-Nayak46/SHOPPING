# 🛒 ShoppingWeb - E-Commerce Platform

An interactive and fully responsive shopping website designed to provide a seamless user experience. Built with core web technologies, this project features dynamic product rendering, advanced filtering, and a modern payment checkout process.

## ✨ Key Features

* **Secure Authentication:** A professional and responsive login/registration page for user access.
* **Dynamic Data Rendering:** Product catalogs are efficiently loaded and managed using JSON.
* **Advanced Product Filtering:** Users can easily find what they need using price range filters and category-specific options.
* **Live Currency Converter:** Real-time currency conversion allowing users to view prices in their preferred local currency.
* **Modern Checkout System:** Integrated QR code payment interface for quick and easy transactions.
* **Payment Backend Simulation:** Includes a robust mechanism for receiving and processing payment data.

## 🛠️ Built With

* **HTML5** - Structuring the web pages.
* **CSS3** - Styling, animations, and responsive design.
* **JavaScript (ES6+)** - DOM manipulation, feature logic, and currency conversion.
* **JSON** - Simulating database storage for products and user data.
